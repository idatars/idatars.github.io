The Chicken McNugget Problem

Generative recursion. Inspired by the classic math proof The Chicken McNugget Problem, this program uses generative recursion to find the optimal combinations of McNugget packs for a given number.

RPN Calculator

Stacks and binary trees. In this implementation of a Reverse Polish Notation calculator, stacks and binary trees are used to compute RPN expressions and convert them into their Infix notation counterparts.

C++

Conway's Game of Life

I/O simulation with observer patterns. This program is an implementation of Conway's Game of Life, the classic zero-player game. Originally done as part of an assignment, it uses observer patterns and simple I/O to continuously generate the next steps of the game.

Flight Planner

Graphs and classic recursion. This is a typical application of simple graph theory that generates all possible flight combinations between two places. It uses text I/O to read-in flight information and print pathways.

Watopoly

UWaterloo themed Monopoly, group project. This is a multiplayer I/O game that is an implementation of a UWaterloo themed version of Monopoly. It was originally coded for a final project, alongside Vishal Chotara and Joe Zhang.
